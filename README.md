# Telecom X - Análise de Evasão de Clientes

## Objetivo
Analisar os dados da empresa Telecom X para identificar fatores que levam ao cancelamento de clientes (churn).

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Etapas da análise
1. Extração de dados de uma API
2. Limpeza e tratamento de dados
3. Análise exploratória (EDA)
4. Visualização de dados
5. Geração de insights sobre churn

## Principais análises
- Distribuição de clientes que cancelaram
- Relação entre tipo de contrato e churn
- Impacto da forma de pagamento
- Tempo de permanência do cliente

## Conclusões
Os resultados mostram padrões importantes que podem ajudar a empresa a reduzir a evasão de clientes.
